# sabaticket
[README.ja.md](README.ja.md)にある日本語のREADMEをご確認ください。

sabaticketはJavaScriptで作られた簡単なチケット管理システムです。ユーザーはQRコード付きのデジタルチケットを発行・使用することができます。

## デモ
[sabaticket 管理パネル](https://code4fukui.github.io/sabaticket/admin.html)で、チケットの作成と管理を行えます。

## 機能
- タイトル、説明、有効期限などのカスタマイズ可能な詳細でチケットを発行
- チケットのQRコードを生成し、スキャンして使用可能
- 発行されたチケットと使用履歴を管理
- チケットに使用される秘密鍵のバックアップと復元

## 要件
このプロジェクトはブラウザ上で動作し、追加ソフトウェアのインストールは必要ありません。

## 使用方法
1. [sabaticket 管理パネル](https://code4fukui.github.io/sabaticket/admin.html)をウェブブラウザで開きます。
2. チケットの詳細を入力し、「発行」をクリックして新しいチケットを作成します。
3. 新しいチケットのQRコードが表示されるので、それをスキャンして使用できます。
4. 「発行済み」セクションに作成したすべてのチケットが表示され、チケットIDをクリックするとQRコードを確認できます。
5. 「秘密鍵のバックアップ」、「秘密鍵の復元」、「秘密鍵の再生成」ボタンを使ってチケットに署名する秘密鍵を管理できます。
